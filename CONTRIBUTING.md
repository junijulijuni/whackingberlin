## Markdown
GitHub works by using Markdown files. Markdown is a specific formatting syntax for text files (similar to rtf, LaTeX, etc). In order for a Markdown page to be readable and look nice it requires the author to have a basic understanding of how to format using Markdown. 

[In-Browser Markdown Editor: Stack Edit](stackedit.io)

[Open Source Markdown Editor for Desktop: MarkText](https://marktext.me/)
