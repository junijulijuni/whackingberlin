## Why Github?
The vision I have for this project is to have an <b> open source, decentralized, wiki-type knowledge base </b> for all things related to whacking berlin. The structure of github,
allowing users to clone the repository onto their local machine ensures that the knowledge shared here is not in the hands of a single person but distributed across the community. Pull requests ideally make it possible for users to collaborate together to add ideas and topics to the wiki. As github is mainly used for code and not something like this, it will take some focused energy to shape this into a usable tool for our community. A final desire I have for this project is to make it visually appealing! Feel free to add emojis, images, gifs, etc. into the topic pages. 
I don't think this is the only way to do this, and if people find other platforms that can fulfill the desired requirements of being open source and decentralized, please feel free to share!
